Connection Issues Detected

Common Cause: This error typically occurs when the target container or service is not running or not responding correctly.

Container Troubleshooting:

  • Check if the container is running:
    docker ps | grep container-name
  • Make sure the container port is correct in your DomainPilot configuration
  • Verify the container's logs for errors:
    docker logs container-name
  • Make sure the container is connected to the domainpilot-proxy network

Local Service Troubleshooting:

  • Verify your local service is running and listening on the configured port
  • Check host-routes.conf has the correct port mapping
  • Try accessing the service directly at localhost:port
